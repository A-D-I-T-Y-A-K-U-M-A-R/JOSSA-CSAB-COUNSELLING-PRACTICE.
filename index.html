FAQs

What is this website about?
This website is a JoSAA and CSAB choice filling simulator that helps JEE aspirants practice counselling before filling their actual choices.

Who can use this simulator?
Any JEE aspirant preparing for JoSAA or CSAB counselling can use this tool to practice choice filling.

Does this website predict admission?
No. This tool is only designed for practice and understanding the counselling process.

Which institutes are included?
The simulator includes IITs, NITs, IIITs and GFTIs participating in JoSAA counselling.

Is this website official?
No. This website is not affiliated with JoSAA, CSAB or Government of India.
